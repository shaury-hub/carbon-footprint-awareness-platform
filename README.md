# Carbon Footprint Awareness Platform

## Project Overview
This platform helps individuals calculate, track, and reduce their carbon footprint using personalized AI-based insights and simple actions.

## Features
- Carbon footprint calculator for travel, energy, and lifestyle.
- Personalized sustainability recommendations.
- Interactive dashboard to visualize progress.

## Tech Stack
- Frontend: React.js
- Backend: Node.js + Express.js (if applicable)
- AI: OpenAI API / Google Gemini API
- Database: MongoDB (optional, if you are saving user data)
